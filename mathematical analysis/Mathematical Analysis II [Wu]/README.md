# Mathematical Analysis II [Wu]
## Overview
&emsp;This is the second part of mathematical analysis in the Chinese mathematics curriculum standards. It furthers the study of single variable calculus and leads into more profound areas in the realm of analysis, particularly series.<br>
&emsp;Also, it should be noted that this book serves as an introduction to the subject.<br>
