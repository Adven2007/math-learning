### goal & comment
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;这次自学概率论是一次尝试。虽然略显青涩，但是无论是知识上还是学习方法上都有诸多收获，所以说是一次宝贵而伟大的尝试。<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;先说知识上，其实没有学到一整套全新的理论，大部分内容在中学接触过。但是，这次 preview 把很多我曾经模糊的认识到的以及零零散散学习过的东西串联起来了，并且初步认识了背后的强大理论体系。但是不得不说，对于更为本质的理论体系，本次 preview 只是浅尝辄止，相信在不久的将来就会忘记。不过，我相信只要努力理解过，这些知识就会在大脑中留有痕迹，哪怕很浅。当再次以更为认真的态度学习概率论时，这条痕迹会被加深，成为灵感之江的河床。<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;再说学习方法上，可以说本次 preview 并非成功，但也不是失败。其优秀之处在于确实效率可观，在加起来三四天左右的时间里，我确实会用概率论的计算方法了，也的确模糊的了解到了概率论基础的框架。不足之处有三。一是，由于学的时候太快，又基本是自己看书、看例题解答，缺乏听课、刷题的过程，导致很可能忘记迅速。二是，本次 preview 让我真的学会了数学上的基础概率论吗？我认为没有。数学是需要一个宏大的理论框架的搭建的，是需要严谨性的，也是需要点缀在逻辑中的灵动的。我可以说这些我都没有体验到。相比之下，我就好比在学习一门工科课程，以较为肤浅的应用为第一目标。三是，我恐怕经常性的这种学习会消磨自己的数学直觉和数学兴趣。我爱数学，不是因为那里有很多知识可以去记住，而是因为那是一个世界，宏大而完善，灵动而有趣。我享受那用直觉去感受的微妙体验，沉醉于那冰火两重天的探索历程，无比怀念那令人悸动的顿悟时刻。如果如此消磨下去，恐怕会丧失这样的能力，让学习数学编程无聊的应付。<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;最后，我想告诉自己，不需要为下学期乃至以后的事情焦虑，永远相信未来自己的强大是现在的自己料想不到的。数学，既然是我无法割舍的白月光，那就为了自己的热爱去学习，而不是任何其他。大可不必为了应付，为了速度，为了比拼去牺牲属于自己的思维升级时间。希望下次 preview 我能更加沉得下心，去体会那数学之美。<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;爱，就是不加解释的付出。
