# MIT 18.200 Principles of Discrete Applied Mathematics
## Syllabus
This is a MIT course for discrete applied math, including counting, generating functions, probability, linear optimization, algebraic structures, basic number theory, information theory, and coding theory, essentially the same as 6.042J, but a little more advanced.<br>
This course is provided with lecture videos, readings, assignments and my notes.<br>
